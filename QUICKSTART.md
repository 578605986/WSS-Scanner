# WSS - Windows Security Scanner
# 快速入门指南

## 项目简介

WSS 是一款专业级Windows安全扫描工具，使用C++和Qt6开发，具有高性能、实时监控、自动隔离等功能。

## 功能特性

- ✅ **多线程文件扫描** - 利用多核CPU加速扫描
- ✅ **实时监控** - 监控文件系统、进程变化
- ✅ **自动隔离** - 加密隔离威胁文件
- ✅ **专业GUI** - Qt6开发的现代化界面
- ✅ **日志报告** - 详细的扫描报告

## 编译步骤

### 1. 安装依赖

#### Visual Studio 2022
- 下载地址：https://visualstudio.microsoft.com/
- 安装组件："使用C++的桌面开发"

#### Qt 6.5+ (MSVC 2019 64-bit)
- 下载地址：https://www.qt.io/download
- 选择版本：Qt 6.5.0 (或更新版本)
- 编译器：MSVC 2019 64-bit

#### CMake 3.20+
- 下载地址：https://cmake.org/download/

### 2. 克隆项目

```bash
git clone https://github.com/yourusername/WSS-Scanner.git
cd WSS-Scanner
```

### 3. 编译项目

```bash
# 创建编译目录
mkdir build
cd build

# 配置（修改Qt路径为您的实际路径）
cmake .. -DCMAKE_PREFIX_PATH="C:/Qt/6.5.0/msvc2019_64"

# 编译
cmake --build . --config Release
```

### 4. 运行

```bash
# 在 build/Release/ 目录下
./WSS.exe
```

## 使用说明

### 快速扫描
1. 点击"快速扫描"按钮
2. 自动扫描常用目录
3. 查看结果列表

### 全盘扫描
1. 点击"全盘扫描"按钮
2. 扫描所有磁盘
3. 等待扫描完成（可能需要较长时间）

### 实时监控
1. 点击"启动监控"按钮
2. 后台自动监控文件变化
3. 发现威胁立即提示

### 自定义扫描
1. 点击"自定义扫描"按钮
2. 选择要扫描的目录
3. 开始扫描

## 添加病毒签名

编辑 `signatures/wss.db` 文件：

```
# 格式: SHA256哈希|病毒名称|风险等级(1-5)
e3b0c44298fc1c149afbf4c8996fb92427ae41e4649b934ca495991b7852b855|Trojan.Win32.Test|4
```

或使用软件界面添加签名。
